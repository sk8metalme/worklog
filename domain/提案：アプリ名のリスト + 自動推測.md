## 提案：アプリ名のリスト + 自動推測

### notify.conf（超シンプル）
```bash
# 対応するターミナルアプリ名のリスト
TERMINAL_APPS=(
  "Ghostty"
  "iTerm"
  "Visual Studio Code"
  "Terminal"
)
```

### notify.sh 側の処理
アプリ名から自動的に検出パターンを推測：
- **TERM_PROGRAM**: 既知のマッピングテーブルで変換（"Ghostty" → "ghostty"）
- **親プロセス**: ワイルドカードで生成（"Ghostty" → "*Ghostty*"）
- **pgrep**: アプリ名をそのまま使用（"Ghostty" → "Ghostty"）

## メリット
- ✅ 設定ファイルは超シンプル（アプリ名を1行ずつ書くだけ）
- ✅ ハードコードではない（設定ファイルで制御）
- ✅ 新しいターミナルを追加するのも簡単

## 制限
- 既知のターミナル（Ghostty、iTerm、VS Code、Terminal）以外を追加する場合、notify.sh のマッピングテーブルの更新が必要になる可能性あり
