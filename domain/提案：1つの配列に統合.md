## 提案：1つの配列に統合

各ターミナルアプリの設定を1行にまとめます。

```bash
# 形式: "TERM_PROGRAM値|親プロセスパターン|pgrepパターン:アプリ名"
TERMINAL_APPS=(
  "ghostty|*ghostty*|ghostty:Ghostty"
  "iTerm.app|*iTerm*|iTerm2:iTerm"
  "vscode|*Code*|Code:Visual Studio Code"
  "Apple_Terminal|*Terminal*|Terminal:Terminal"
)
```

これで：
- 設定ファイルが1つの配列だけになる
- 1行で1つのターミナルの全情報を管理
- 追加・削除が簡単
