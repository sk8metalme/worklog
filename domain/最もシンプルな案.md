## 最もシンプルな案

```bash
# 形式: "TERM_PROGRAM値:アプリ名"
TERMINAL_APPS=(
  "ghostty:Ghostty"
  "iTerm.app:iTerm"
  "vscode:Visual Studio Code"
  "Apple_Terminal:Terminal"
)
```

**これだけ**で、notify.sh 側が自動的に：
- 親プロセス名パターン: `*Ghostty*` のように自動生成
- pgrep パターン: アプリ名をそのまま使用

## メリット
- ✅ 設定は2つの値だけ（TERM_PROGRAM値とアプリ名）
- ✅ 人間が読みやすい
- ✅ 新しいターミナルを追加する時も簡単
